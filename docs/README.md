# OpenTabSsh 文档索引

## 新接手必读顺序

1. `AGENT_HANDOFF.md`：当前事实、边界和第一执行序列。
2. `WORKSPACE_PATHS.md`：唯一允许的构建、测试、日志、备份和清理路径。
3. `PROGRESS.md`：已实现与未实现功能。
4. `CORE.md`：Mock/真实 Native Core 架构和安全要求。
5. `BUILD_TEST.md`：构建、设备验证和单次完整验收规则。
6. `ISSUES.md`：已知问题与风险。
7. `FILES.md`、`UI.md`：文件职责与页面结构。
8. `UPSTREAM_REFERENCES.md`：TabSSH Web/Android/Desktop 上游参考源码。
9. `GIT_PUBLISH.md`：提交、推送和 Release 验收。
10. `NEW_CHAT_PROMPT.md`：新对话可直接复制的提示词。

历史/专项资料：`ANDROID_TO_HARMONY_MAPPING.md`、`IMPLEMENTATION_STEPS.md`、`LIBSSH2_COMPILE_GUIDE.md`、`NATIVE_CORE_FUNCTIONS.md`。

任何功能、路径、构建、测试或发布状态变化，都必须同步更新上述相关文档。真实功能只能以端到端证据标记完成。
