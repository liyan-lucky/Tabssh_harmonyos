# UI 页面说明

## 图标来源

所有应用图标统一使用 ProIcons 资产，禁止自绘 SVG 和 Emoji/Unicode 字符图标。底栏沿用 RustDesk HarmonyOS 的悬浮胶囊布局，但图标资源分别使用 ProIcons tune/network/monitor/display；页面内图标映射、来源和审计规则见 `PROICONS_ICONS.md`。

## 现有样式规则

- 首页继续使用圆角卡片、轻阴影、蓝色主操作和安全区内悬浮胶囊底栏；工作台、设置 Tab、设置页、工具箱页、底部 Tab 胶囊、连接编辑、终端、SFTP、端口转发和已迁移二级页已接浅色/深色两套主题色。
- 应用使用全屏窗口；所有已注册 ArkUI 页面根容器必须读取系统/挖孔/手势避让区，顶部内容、滚动内容、底部控件和悬浮胶囊都不能被状态栏、导航栏或手势区域遮挡。
- 顶部 Logo/标题区采用 RustDesk HarmonyOS 风格的多段半透明渐变：暗色基底 `#171A1E` 从 `#BB171A1E` 过渡到 `#00171A1E`，亮色基底 `#F0F4FA` 从 `#C0F0F4FA` 过渡到 `#00F0F4FA`；默认首屏必须与状态栏分开，`HeaderOverlay` 使用 `headerStatusInset() + 96`，状态栏占位使用 `Math.max(24, avoidStatusBarHeight - 18)`，内容顶部从 `Math.max(80, headerOverlayHeight() - 40)` 起步，普通 Header 行高 80、监控 Header 行高 92，允许第一张卡片进入渐变尾部但不能遮挡状态栏、Logo 或标题。滚动后的内容位置不能作为默认首屏间距依据。
- 底部 Tab 区采用半透明材质色与 Thin blur：浅色 `#30FFFFFF`、暗色 `#20000000`，边框分别为 `#28FFFFFF` / `#18FFFFFF`，内容区可以滚动到胶囊下方但必须保留底部避让；全屏覆盖层的空白区域必须透明命中，不能阻断中间内容滚动或点击。
- 工作台右上角入口为“工具箱”，不再跳转系统设置；工作台主机区域只展示一张主机管理卡片，左侧点击进入已保存主机独立页面，右侧添加按钮进入新增主机，卡片下方不再显示新增主机/已保存主机/连接历史三行入口，也不显示已保存主机的名称、地址、用户、端口或状态。连接 Tab 显示最近 10 条历史摘要。
- 连接页新增搜索、搜索命中高亮、收藏筛选、分组筛选、排序芯片和批量操作时，不改变整体页面结构；logo 下方不再放 SSH/二进制横幅，页面应直接从顶部避让区进入“连接方式”；筛选卡片、输入底色、芯片和阴影必须使用 `AppTheme` token，不能写死浅色灰。
- 连接编辑页已接主题色和中英双语，保留基础字段编辑、密码/私钥切换、私钥导入清理和所属分组芯片选择，不引入新图标体系。
- 连接分组页沿用浅蓝背景、白色圆角卡片、轻阴影、文字按钮和小色块标识，不引入新图标体系。
- 访问日志页沿用浅蓝背景、白色圆角卡片和 ProIcons timer 图标；只展示本地摘要日志，不展示命令输出、密码、私钥口令或服务器地址。
- 导入导出页沿用浅蓝背景、白色圆角列表和现有 ProIcons 资源；导出不包含密码、私钥文件、私钥口令或命令输出。
- 工具箱页使用现有 ProIcons rawfile 资源，不新增自绘 SVG；JSON、编码、文本、颜色、单位、系统信息、存储摘要、IP 基础信息、公网 IP、受控子网发现、访问审计、默认网络/DNS/网关/路由/地址族摘要、TCP 连通性、ICMP 等价验收、端口扫描、HTTP 下载/上传测速、Nginx 配置摘要/同输入变量展开/include 检出和 QR Version 2-L 矩阵、二维码预览和 SVG 保存选择器已有同页工具面板；二维码 SVG 真实目标写入/回读、PNG/相册保存、扫码兼容性和进一步美化、IP 详情路由/地址族设备点击、多网卡枚举和外部 Nginx include 文件导入/展开仍待补。
- 设置 Tab 直接展开浅色/深色和语言分段切换；语言支持“系统 / 中 / EN”，系统选项通过 `@ohos.i18n` 读取首选系统语言并归一到中文/English。偏好通过 HarmonyOS preferences 持久化，并驱动首页、设置、工具箱、关于、终端设置、连接历史、访问日志、连接分组、连接导入导出、连接编辑、终端、SFTP 和端口转发页面刷新。
- 收藏状态用文字标签呈现，不引入 Emoji/Unicode 图标。
- 筛选、排序、收藏、分组已绑定 RDB-backed 仓库；未取得跨重启点击证据前不能写成完整持久化验收通过。
